# replace_with_antonym
Juman辞書に含まれている反義語の情報を利用して、文中に現れる反義語を全て置換する。

## Usage
    $cat input.txt
    海は広い。
    $cat input.txt | juman | python src/replace_with_juman.py
    海は狭い。

## 既知の不具合
ナ形容詞特殊のように、「ダ列基本連用形」「デアル列基本連用形」など複数の活用形を持つものは変換できない
